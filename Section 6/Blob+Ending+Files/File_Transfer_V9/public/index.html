File Transfer With WebRTC

Transfer files to another user in real time through your browser or phone.


Getting started

  1. The file is split into chunks which are then transferred via a WebRTC RTCDataChannel.
  2. Send and receive progress is monitored using the HTML5 progress element.
  3. At the receiver end, the file is reassembled using the Blob API and made available for download.
